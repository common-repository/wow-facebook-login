Social Login for WordPress via Facebook account. Highly customizable button 'Login with Facebook'. 

== Description ==
Social Login for WordPress provides the ability to easily integrate the button 'Login with Facebook' on the website. Let visitors easily authorize on your WordPress website with their Facebook account.

[Demo](https://wow-estore.com/item-preview/wow-facebook-login-pro/)

= Features =
* Upon initial login, the button creates a user role for the user that you have set in your WordPress settings as the 'new user default role': subscriber, contributor, author or administrator
* Members are saved into a built-in list with their data available for you: email, first name, last name
* Powerful styling
* Customizable texts
* Option to redirect to a URL of your choice after the login via Facebook
* Option to redirect to a URL of your choice after the initial login specifically
* Button also appears on the wp-admin login page

= Pro version =
* Send an notification to every new user
* Stream the new emails automatically to MailChimp, Getresponse, Activecampaign, Aweber and Sendinblue for email marketing
* Send an custom email to every new user
* Option to hide the topbar
* Compatible with WooCommerce
* Compatible with EDD
* Export users info in CSV file

== Installation ==

* Installation option 1: Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
* Installation option 2: Download the zip file, then upload the plugin via the wp-admin in the `Plugins` -> `Add new` section. Or unzip the archive and upload the folder to the plugins directory `/wp-content/plugins/` via ftp
* Press `Activate` when you have installed the plugin via dashboard or press `Activate` in the in the `Plugins` list 
* Go to `Wow Facebook Login` section that will appear in your main menu on the left
* Check the Readme section in the plugin and follow those instructions to create your Facebook app, needed for this plugin to work
* Enter your App ID and your App Secret in the plugin's settings
* Customize the button as you wish
* Copy and paste the shortcode [Wow-Facebook-Login] to where you want the button to appear. 
* If you want it to appear everywhere on your site, you can insert it for example in your `header.php`, like this: `<?php echo do_shortcode('[Wow-Facebook-Login]');?>`
* If you want to insert the shortcode into a widget, use <a href="https://wordpress.org/plugins/shortcode-widget/">this plugin</a>
